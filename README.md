# Comandos:
 - comando de MS - DOS:
   - CD..
   - Mkdir 'Nome da pasta'
 - comando de Git:  
   - Git Init
   - Git Add
   - Git 
- comando do bash:
   - node -v => verificar a versão de  node instalada 
- comando de Cucumber:
   - Cucumber --init => Para executar todos testes
   - cucumber --init -t @v1 => Para executar  apenas os casos de testes com a tags
